О проекте

Привет, рады видеть вас на станице нашего домашнего проекта. Поводом для его создание послужило желание попробовать себя в роли backend разработчика. Пройдя курсы, решил испробовать на практике знания фреймворка Django. Рустам решил заняться frontend частью интренет магазина, для обретения новых навыков. Данная страница полностью реализована на Django template и bootstrap. Интернет магазин же реализован по принципу BFF- "backend for frontend". В разделе блог появятся статьи с описанием примен
